Guide d'nstallation



Pour installer le modpack vous avez premièrement besoin de votre copie légalement obtenue du jeu de base Mario Kart 7 soit en format cartouche soit installée sur votre système

Ensuite installez le fichier CIA du modpack soit sur Gamebanana avec FBI soit utilisant l'Universal Updater

Puis lancez le launcher CTGP-7 depuis le menu home et suivez les instructions à l'écran pour finaliser l'installation


Pour installer le modpack vous avez premièrement besoin de votre copie légalement obtenue du jeu de base Mario Kart 7 ainsi que votre mise à jour 1.2 légalement obtenue au format CIA en suivant ce guide

Ensuite installez le fichier CIA du modpack trouvé sur Gamebanana et le jeu + sa mise à jour sur Citra utilisant l'option Fichier > Installer un fichier CIA

Puis lancez le launcher CTGP-7 depuis le menu de Citra et suivez les instructions à l'écran pour finaliser l'installation
