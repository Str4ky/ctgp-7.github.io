Description



CTGP-7 est un modpack pour le jeu Mario Kart 7 sorti sur la console Nintendo 3DS le 1er décember 2011 qui inclut :


Le modpack rajoute plus de 20 coupes customisées qui font un total de plus de 80 courses customisées comme :

  • CTR Château de Cortex

  • Glissade secrète

  • Île tuyaux

  • Route arc en ciel Deluxe

  • Spectacle Miku


Le modpack rajoute plus de 20 personnages personnalisés comme :

  • Sonic

  • Kirby

  • Sac eshop

  • Funky Kong

  • Waluigi


Le modpack rajoute plus de 20 pièces de véhicules pouvant être utilisées pour customiser votre kart en jeu


Le modpack rajoute de nouveaux objets venant d'autres jeux comme :

  • Champignon géant

  • Fausse boîte


Le modpack de nouveaux modes de jeu customisés comme :

  • Mode VS

  • Mode mission

  • Compte à rebour

Mode CTWW



CTWW (pour Courses Customisées Mondiales en Anglais) est une fonctionnalité de CTGP-7 permettant aux utilisateurs du mod de jouer à des pistes personnalisées en ligne. CTWW est activé par défaut et peut être accédé simplement en allant en ligne !


Afin de distinguer votre console des autres, un ID de console unique est utilisé pour toutes les communications avec le serveur CTGP-7.


Cet ID est composé de 0x plus 16 symboles de "0" à "9" et de "A" à "F". L'ID est un hash dérivé d'une série d'autres ID uniques de console liés à votre puce NAND (comme il s'agit d'un hash, il ne peut pas être utilisé à l'envers pour obtenir vos ID privés).


Vous pouvez considérer l'ID de la console CTGP-7 comme votre "code ami" spécifique pour le mod.


Vous pouvez obtenir votre ID de console CTGP-7 à partir du menu en jeu.
