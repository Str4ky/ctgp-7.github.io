Installation guide



To install the modpack you first need your legally obtained base Mario Kart 7 game in either cartridge format or installed on your system

Then install the CIA file of the modpack either on Gamebanana with FBI or using the Universal Updater

Finally launch the CTGP-7 Launcher from the home menu and follow the onscreen instructions to finish the installation
